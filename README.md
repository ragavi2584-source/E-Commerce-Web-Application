# E-Commerce-Web-Application
An E-Commerce Web Application is an online shopping platform where users can browse products, add products to their cart, place orders, and track order status. The application includes user authentication, product management, and order management features.

This project is a simple online shopping application.

## Technologies Used
- HTML
- CSS
- JavaScript


## Features
- Product Catalog
- Add to Cart
- Checkout Page
- Responsive Design

## Future Enhancements
- Database Integration using MySQL
- User Authentication
- Order Tracking
